\doxysubsection{Detailed Description}
Class for interfacing with a Reed Switch using the ESP32\textquotesingle{}s Pulse Counter (PCNT) peripheral. 

This class configures a GPIO pin as an input for the Reed Switch, sets up a PCNT unit and channel to count pulses (e.\+g., for rotation or events), and handles configurations like debouncing and event limits.
