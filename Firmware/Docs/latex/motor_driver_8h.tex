\doxysubsection{Detailed Description}
Definition of the \doxylink{class_motor_driver}{Motor\+Driver} class for controlling DC motors using the ESP32. 


\begin{DoxyItemize}
\item This class encapsulates the logic to control the direction and speed of a DC motor via an L6205 H-\/bridge driver. It uses the ESP32\textquotesingle{}s LEDC peripheral (PWM) for speed control (Duty Cycle) and GPIO pins for direction control. 
\end{DoxyItemize}
