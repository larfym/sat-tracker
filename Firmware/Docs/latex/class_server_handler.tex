\doxysubsection{Detailed Description}
Manages the ESP32\textquotesingle{}s asynchronous web server, handles API requests, and serves configuration/\+status data. 

This class sets up the HTTP server, defines endpoints for data retrieval, configuration saving, manual control, and tracking status toggling.
