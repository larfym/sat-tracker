\doxysubsection{Detailed Description}
Class for current reading using an ESP32 ADC channel. 

This class configures an ADC channel, calibrates the conversion to millivolts (mV), and calculates the current in milliamperes (mA), considering the shunt resistor and the amplifier gain.
