\doxysubsection{Detailed Description}
Discrete Proportional–\+Integral (PI) controller implementation. 

This class implements a discrete-\/time PI controller using the incremental (velocity) form of the control equation. It is designed for embedded control applications such as motor control in the satellite tracking system.

The controller supports:\+
\begin{DoxyItemize}
\item Adjustable proportional and integral gains
\item Output saturation
\item Dead-\/zone around the error
\item State reset
\end{DoxyItemize}

The controller internal equation follows:\+

\[u[k] = u[k-1] + a\,e[k] + b\,e[k-1]
\]

where:\+
\begin{DoxyItemize}
\item $e[k]$ is the current error
\item $u[k]$ is the control output
\item $a$ and $b$ are coefficients derived from PI gains
\end{DoxyItemize}

This implementation is suitable for real-\/time control loops running at fixed sampling periods.
