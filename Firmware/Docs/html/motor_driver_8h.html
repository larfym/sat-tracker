Detailed Description

Definition of the MotorDriver class for controlling DC motors using the ESP32.

  • This class encapsulates the logic to control the direction and speed of a DC motor via an L6205 H-bridge driver. It uses the ESP32's LEDC peripheral (PWM) for speed control (Duty Cycle) and GPIO pins for direction control.
