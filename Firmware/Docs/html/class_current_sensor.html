Detailed Description

Class for current reading using an ESP32 ADC channel.

This class configures an ADC channel, calibrates the conversion to millivolts (mV), and calculates the current in milliamperes (mA), considering the shunt resistor and the amplifier gain.

void CurrentSensor::calibrateOffset ( )

Calibrates the sensor's voltage offset.

Measures the voltage of the ADC pin when the input current is zero (0) to establish the offset_mV value.

Warning
Must be called with no current applied to the motor.
