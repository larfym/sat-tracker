◆ __attribute__()

struct __attribute__ ( (packed) )

Status flags for the satellite tracking system.

Represents the angular position of the antenna.

Uses bit-fields to pack multiple status indicators and a 3-bit error code into a single byte.

< True if TLE data has been successfully loaded.

< True if GPS has a valid fix.

< True if antenna is in manual control mode.

< True if movement is active.

< internal error code.

< Reserved bits for future use.

< True if the mount has completed its homing procedure.

< True if the stop task is done

< True if the next pass prediction has been

< True if a system reset is pending.

< True if antenna offsets have been updated.

< True if TLE data has been updated.

< Reserved bits for future use.

< Azimuth angle in degrees (0–360).

< Elevation angle in degrees (0–90).
